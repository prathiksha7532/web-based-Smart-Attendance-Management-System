# 🎓 Smart Attendance Management System Using Face Recognition

## 📌 Project Overview
The Smart Attendance Management System is a web-based application that uses Face Recognition to automate student attendance.

## ✨ Features
- 🔐 Secure Login
- 👤 Student Registration
- 👨‍🎓 View Registered Students
- 📷 Face Attendance
- 📋 Attendance Records
- 🤖 AI Face Detection using face-api.js
- 🌐 Hosted on GitHub Pages

## 💻 Technologies Used
- HTML5
- CSS3
- JavaScript
- Face API.js
